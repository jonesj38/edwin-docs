# EdwinPAI

<p align="center">
  <img src="/assets/edwin-logo.png" alt="EdwinPAI" width="220" />
</p>

EdwinPAI is a personal AI runtime that helps you run agents across chat, local tools, and connected devices.

## What it is

- A gateway for messaging, browser, device, and local-tool automation
- A runtime for persistent agent sessions and workflows
- A practical way to run your own AI stack on hardware you control

## Quick install

```bash
curl -fsSL https://edwinpai.com/install.sh | bash
```
