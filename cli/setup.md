# `edwin setup`

Initialize `~/.edwin/edwin.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
edwin setup
edwin setup --workspace ~/.edwin/workspace
```

To run the wizard via setup:

```bash
edwin setup --wizard
```
